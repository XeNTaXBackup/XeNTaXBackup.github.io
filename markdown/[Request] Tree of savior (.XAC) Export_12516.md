Hi
Here is another importer for models and animations from this game.
It requires Blender 249 and Python26.
Game don't use for npc and pc full head and hair meshes but textured plane like avatar.
For right position of this heads and hairs first import body.
After that import "hair" and "head" file and push play animation button. 
Go to Blender Outliner and find 'hair-mesh' or 'head-mesh' and change visibility.

For some pc heads must manual change uv map in Blender UV Editor

Sample
